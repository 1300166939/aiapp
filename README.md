# Test Knowledge Report Studio

一个面向自动化测试失败诊断的 Codex 报告生成 APP。

它帮助测试同学把失败上下文、证据链、初稿-运行-纠偏-更新-复验过程、人机边界整理成一份可提交的 Markdown 报告。

## 运行

```bash
npm start
```

打开：

```text
http://127.0.0.1:4173
```

## 测试

```bash
npm test
```

这个 APP 来自自动化测试平台知识库建设场景，适合沉淀：

- 失败诊断报告
- 证据链
- 人工确认项
- Codex 可处理范围
- 复验记录
